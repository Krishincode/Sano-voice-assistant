Sano is a Python-based voice assistant that performs basic tasks using voice commands. It listens through the microphone, processes speech, and responds using text-to-speech.

## Features
- Opens YouTube and Google
- Opens Visual Studio Code
- Plays music from local folder
- Voice-based interaction
- Exit using voice command

## Technologies Used
- Python
- SpeechRecognition
- pyttsx3
- Webbrowser
- OS module

## How to Run
1. Install dependencies:
   pip install -r requirements.txt

2. Run the program:
   python sano.py

## Note
- Internet connection is required for speech recognition
- Microphone access must be enabled
- Paths for VS Code and Music folder may need to be updated
